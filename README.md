# gundamStore
eCommerce App to purchased gundam



## Usage
---
Rename .env.sample to .env and add requires environment variables. Using port 3000 for frontend and 4000 for backend

### Install dependencies
```
#Backend deps
npm install

#Frontend deps
cd frontend
npm install
```

### Run server
```
cd api
npm start
```

### Run client
```
cd client
npm start
```
